# ProductFlow Static

ProductFlow is a learning frontend project.

The goal of this version is to build the first static HTML structure for a product management dashboard.

## Current version

ProductFlow Static v0.1

## How to open

Open `index.html` in a browser.
Or use VS Code Live Server:

1. Open the project folder in VS Code.
2. Right-click `index.html`.
3. Choose `Open with Live Server`.

## Pages

- Landing page: `index.html`

## What I practiced

- Basic HTML document structure
- Semantic tags
- Navigation
- Sections
- Lists
- Buttons and links
- Basic accessibility structure

## Limitations

- The project is static.
- The CTA button does not have JavaScript behavior yet.
- Styles are not implemented yet.

# ProductFlow Static

ProductFlow is a learning frontend project.

The goal of this version is to build the first static HTML structure for a product management dashboard.

## Current version

ProductFlow Static v2.3 JavaScript Loops

## Pages

- Landing page: `index.html`
- Create project page: `create-project.html`
- Dashboard page: `dashboard.html`

## How to open

Open any page directly in a browser:

- `index.html`
- `create-project.html`
- `dashboard.html`

Or use VS Code Live Server:

1. Open the project folder in VS Code.
2. Right-click the HTML file.
3. Choose `Open with Live Server`.

## Limitations

- The project is static.
- JavaScript behavior is not implemented yet.
- Project details pages are not implemented yet.
- Form submission does not create a real project yet.

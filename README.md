# Frontend Learning

This repository contains my frontend learning process.

## Goal

The goal is to learn frontend development step by step and build portfolio-ready projects using HTML, CSS, JavaScript, React, TypeScript, and Next.js.

## Main project

ProductFlow is a learning product management dashboard.

Current stage:

- ProductFlow Static v2.3 JavaScript Loops
- HTML and CSS baseline completed
- JavaScript variables, functions and arrays practiced
- for loops and value accumulation practiced
- Ready for objects

Project folder:

- `productflow-static/`
